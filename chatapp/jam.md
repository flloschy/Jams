# Chatapp
Everyone creates their own kind of client to chat with all the others over a single backend. Some might create a website others maybe a TUI.
###### Screw the singular backend xd


### How to Use
- clone
- `npm i`
- `npm run dev`
    - or
    - `npm run build`
    - `npm run preview`
